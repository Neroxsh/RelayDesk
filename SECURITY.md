# Security Policy

## 报告安全问题

请不要在公开 issue 中发布可利用细节、设备密钥、配对码或会话内容。先通过 GitHub 私密安全报告联系维护者，并附上受影响版本、复现条件和可能影响。

## 安全边界

- 电脑端不监听公网地址，只在 `127.0.0.1` 提供控制中心。
- 中继只传递端到端加密消息，不应获得设备密钥或会话明文。
- 新设备需要在电脑本机确认。
- 远程请求限制为项目定义的会话操作，不接受任意 shell 命令。

如果怀疑设备凭据泄露，请立即在电脑控制中心解除对应设备，并删除 `%USERPROFILE%\.relaydesk\config.json` 后重新配对。删除配置也会移除全部已有绑定。
